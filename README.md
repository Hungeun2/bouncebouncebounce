# Bounce Blocks

모바일 웹에서 바로 플레이할 수 있는 공튀기기 게임 프로토타입입니다. 의존성 없는 정적 웹앱이라 `index.html`만 배포해도 동작합니다.

## Run locally

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

## Controls

- Mobile: left / right / BOOST buttons
- Desktop: arrow keys or A/D, Space/W/Up for boost, R to restart

## Deploy

This folder can be deployed as a static site on Vercel, Netlify, GitHub Pages, or Cloudflare Pages. Use the project root as the publish directory.
